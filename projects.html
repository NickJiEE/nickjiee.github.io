8-bit Kogge-Stone Adder

A high-speed 8-bit Kogge-Stone Adder (KSA) using static CMOS logic, designed and simulated in Cadence.

8-bit Kogge-Stone Adder

This project implements a high-speed, 8-bit Kogge-Stone Adder (KSA) implemented using static CMOS logic in Cadence, designed for minimum delay, reduced transistor count, and robust signal integrity at the transistor level.

🔍

The design achieves logarithmic delay and is well-suited for large-scale arithmetic circuits where speed is critical.

Cosmic Whisper

An educational game for people interested in optics and circuits, as part of research supervised by Professor Saharnaz, UCSD.

Cosmic Whisper

Cosmic Whisper is an interactive educational game developed to inspire interest in optics and electrical engineering through immersive and hands-on problem solving. Designed under Professor Saharnaz’s supervision at UC San Diego, the game casts players as scientists trying to communicate with an alien civilization orbiting a chaotic three-star system, inspired by The Three-Body Problem.

🔍

Packaged in a durable stainless steel housing and powered by an ESP32-S3 microcontroller, the game unfolds through a multi-stage puzzle sequence that blends real-world experiments with digital logic. Players tackle challenges involving light transmission and absorption, spectroscopy, optical ciphers (Morse code and Vigenère), and light intensity modulation.

🔍

For more details, check out the GitHub repository!

Modular Power Hub (IP)

A dual-input, multi-output automatic power selector for microcontrollers and small electronics with selectable voltage rails.

Modular Power Hub

The Modular PowerHub was designed to provide a flexible, reliable power solution for a variety of electronics projects. It allows seamless switching between multiple input sources, such as USB-C, USB-A or barrel jack adapters, while automatically preventing backfeed or voltage conflicts. By supporting multiple output voltage rails, it can power different devices with multiple voltage options.

🔍

The board uses a TPS2115A dual-input power-path controller to manage input selection, combined with decoupling capacitors, a polyfuse, and a TVS diode to protect connected devices. From the common 5V output, the board provides multiple regulator paths including 3.3V and optional adjustable voltage rails. Modular headers make it easy to connect microcontrollers, sensors, or other electronics, making the design versatile for prototyping and embedded systems.

🔍

For more details, check out the GitHub repository!

SonicCar

An intelligent voice-controlled autonomous vehicle using LLMs & object detection.

SonicCar

In recent years, autonomous vehicles have advanced rapidly, driven by breakthroughs in artificial intelligence (AI), sensor technologies, and deep learning. As part of UCSD’s ECE/MAE148 (Introduction to Autonomous Vehicles) course, our team developed SonicCar, a state-of-the-art autonomous vehicle capable of responding to voice commands and navigating intelligently and safely.

🔍

SonicCar integrates speech-to-text recognition with a large language model (LLM), enabling it to interpret and execute natural language commands. To ensure safe operation, it combines LiDAR-based obstacle detection with a camera-based stop sign recognition system trained using RoboFlow deep learning. Together, these systems enhance reliability, situational awareness, and user experience in real-world driving conditions.

🔍

For more details, check out the GitHub repository!

WearWise

WearWise, a smart IoT Wardrobe Management App

WearWise

WearWise (IoT Wardrobe Management App) is a secure platform that lets users monitor personal environmental sensing devices and manage their wardrobe efficiently. Through a robust authentication system, users can register ESP32 devices, track real-time temperature and humidity data, and visualize trends on an interactive dashboard. The app also features an AI-powered assistant that provides outfit recommendations based on weather forecasts, while users can maintain a personalized wardrobe by adding, updating, or removing clothing items.

🔍

Key Functionalities:
• Secured User Info: Passwords are hashed and user data is protected in a secured database.
• User-Devices-Wardrobe Relationship: Users can customize device and clothing names, track their wardrobe, and update status in real time.
• Adding Devices via QR-Code: Quickly register sensors by scanning QR codes.
• Real-Time Sensor Data Display: Monitor live temperature readings from ESP32 devices with charts.
• Real-Time Weather Checking: Get localized weather forecasts to assist with outfit suggestions.
• Smart AI Secretary: Provides clothing recommendations based on wardrobe and environmental data.
• Two-Factor Authentication: Optional 2FA for enhanced account security.

🔍

For more details, check out the GitHub repository!

Bitcoin Hashing

Bitcoin hashing using SHA-256 as a cryptographic process that secures transactions on the Bitcoin network.

Bitcoin Hashing

This project demonstrates the implementation of Bitcoin's proof-of-work mechanism by simulating the SHA-256 hashing process used in Bitcoin mining. The design is implemented in SystemVerilog and includes various configurations (serial and parallel) to explore different hashing methods.

🔍

This project serves as an educational tool to understand the cryptographic processes behind Bitcoin mining, focusing on the SHA-256 algorithm's role in securing transactions and maintaining the blockchain's integrity.
